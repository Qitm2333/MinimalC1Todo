# 📋 极简待办 - MinimalTodo

一个简洁、优雅的本地待办事项应用，专注于提升效率而非功能堆砌。

## ✨ 特性

### 核心功能
- ✅ 添加、编辑、删除任务
- ✅ 标记任务完成状态
- ✅ 本地存储（数据永久保存）
- ✅ 实时搜索任务
- ✅ 优先级管理（高/中/低）
- 🎨 **任务颜色标签** - 6种颜色自定义标记任务
- 💾 **数据备份** - 一键导出/导入JSON数据
- 🌓 **暗黑模式** - 舒适的夜间主题

### 智能分类
- 📝 **全部任务** - 查看所有待办事项
- ☀️ **今日任务** - 专注当天要完成的任务
- ⭐ **重要任务** - 高优先级任务快速访问
- ✅ **已完成** - 查看完成记录

### 专注模式 🎯
- ⏱️ **计时器** - 精确记录专注时间
- 🌈 **彩虹流光** - Apple风格的沉浸式界面
- 📊 **时间统计** - 24小时可视化时间线
- 🎨 **多层光效** - 旋转背景光晕 + 边框流光

### 数据统计与日历
- 📈 总任务数量与完成率
- 📅 **日历面板** - 月度视图
- 🗓️ **多日期支持** - 一个任务可拖放到多个日期（适合跨天任务）
- ⏰ **24小时时间线** - 每日专注记录可视化
- 🎨 任务颜色在时间线中显示
- ⏱️ **专注时长统计** - 任务卡片实时显示累计专注时间
- 🗑️ 单条专注记录删除功能
- ✂️ 多日期任务支持单独移除某个日期

### 布局定制
- 📌 **可收起侧边栏** - Tab键或按钮快速切换
- 📏 **可调整宽度** - 拖动分隔条自定义布局
- 📱 **竖屏适配** - 移动设备专属优化布局

## 💡 使用技巧

### 快捷操作
- 输入框按 `Enter` 快速添加任务
- 编辑模态框按 `Enter` 保存修改
- 点击复选框快速标记完成
- 悬停任务卡片显示操作按钮
- 按 `Tab` 键快速收起/展开任务列表
- 右上角月亮/太阳图标切换暗黑模式

### 专注模式使用
1. 点击任务卡片右侧的"专注"按钮
2. 进入全屏专注界面，欣赏彩虹流光效果
3. 点击"开始"计时，"暂停"可随时暂停
4. 点击"结束"保存本次专注记录
5. 在日历面板查看24小时时间线统计
6. 任务卡片会自动显示累计专注时长

### 多日期任务管理
1. 拖动任务卡片到日历上的日期格子
2. 可以将同一任务拖到多个日期（适合多天完成的任务）
3. 任务卡片会显示"X个日期"提示
4. 悬停日历中的任务项，点击 × 可从单个日期移除
5. 适用场景：画插画、写文章等需要多天持续的任务

### 优先级建议
- **高（红色）** - 紧急重要的任务
- **中（橙色）** - 重要但不紧急的任务
- **低（绿色）** - 日常待办事项

### 颜色标签使用
- 🔵 **蓝色** - 工作相关
- 🟢 **绿色** - 学习成长
- 🟠 **橙色** - 生活杂务
- 🔴 **红色** - 紧急事项
- 🟣 **紫色** - 兴趣爱好
- 🩷 **粉色** - 健康运动

### 最佳实践
1. 每天早晨规划今日任务
2. 设置合理的优先级和颜色标签
3. 跨天任务拖到多个日期，合理分配工作量
4. 使用专注模式提高效率，关注累计时长
5. 定期导出数据备份
6. 查看时间线回顾每日专注情况
7. 定期清理已完成任务
